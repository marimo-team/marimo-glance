# Security Policy

marimo Glance renders marimo notebooks inline on code-hosting sites. Notebook source is compressed into the page URL fragment and run by the in-browser WASM runtime, so it never leaves your machine.

## Reporting a Vulnerability

Found something? [Open a private advisory on GitHub](https://github.com/marimo-team/marimo-glance/security/advisories/new) or email security [at] marimo [dot] io. A description and steps to reproduce are all we need.

We'll acknowledge within 3 business days and keep you posted, and we're happy to credit you in the advisory.

## Scope

This covers the marimo Glance extension and the packages in this repo. For marimo itself, the marimo.app playground, or molab, use the [marimo security policy](https://github.com/marimo-team/marimo/security).
